Quick Start Guide.

Step 1

Download the h2zero jar file

Download the required dependencies

Step 2

Create your .h2zero file

To get started quickly, you can use the following h2zero file as a reference point

Step 3

Set up your ant build.xml file

<path id="classpath-h2zero">
	<fileset dir="lib/runtime">
		<include name="*.jar"/>
	</fileset>
</path>
<target name="h2zero" description="h2zero generate">
	<taskdef name="h2zero" classpathref="classpath-h2zero" classname="synapticloop.h2zero.ant.H2ZeroTask" />
	<h2zero in="src/main/java/your-h2zero-file-name-here.h2zero" outDir="." verbose="false" />
</target>

Step 4

Generate the code!

ant h2zero

This will generate all of the required code for you h2zero file and database creation scripts
