Step 3

Set up your ant build.xml file 

				
<path id="classpath-h2zero">{\n}
{\t}<fileset dir="lib/runtime">{\n}
{\t}{\t}<include name="*.jar"/>{\n}
{\t}</fileset>{\n}
</path>{\n}

<target name="h2zero" description="h2zero generate">{\n}
{\t}<taskdef name="h2zero" classpathref="classpath-h2zero" classname="synapticloop.h2zero.ant.H2ZeroTask" />{\n}
{\t}<h2zero in="src/main/java/your-h2zero-file-name-here.h2zero" outDir="." verbose="false" />{\n}
</target>{\n}

Step 4

Generate the code! 

				
ant h2zero

This will generate all of the required code for you h2zero file and database creation scripts
